第十五章 其实是十二

“Nohm，你怎么从德国回来了天天抱着手机玩？”助理在拍戏间隙问道。

Nohm还没从喜悦中缓回来，咧着张小脸笑道：“没有啊，Pat哥。”

Pat哥捏着兰花指：“别叫哥了，叫姐。”说着她撩了一下自己新做的假胸，随即做作的摸了摸自己的刘海：“我说你这个手机就是那天让我拿去修的那个吗？”

“嗯，对。”他点了点头，脸上还带着笑。

Pat拍了拍他的肩膀：“姐看你这样子跟最开始和Vie恋爱的时候一模一样，但姐观察了几天，Vie那边并不怎么开心。姐提醒你一句，合约还没到期，这部戏又在拍，你最好不要惹出什么幺蛾子。”

他的笑容慢慢融化，眼神漠然：“知道了姐，我不会耽误工作的。”

而Vie看他心情不错，想着他应该是恢复过来了，于是自己又端起了架子，等着Nohm主动找自己搭话，买好吃的来哄她。

没想到，工作之外再见Nohm时，他身边竟然多了一个挑不出半点优点的异国女生。

危机感顷刻来袭。

眼前的少年正以冬雪消失在春日的势头，默不作声地离她远去。可这是泰国，泰国没有冬天，公主也从未经历过失去。

Vie把那封莫名其妙的信撕得粉碎，趴在床上放声大哭。

什么狗屁恋人捆绑，什么狗屁豢养的鱼，说得好像跟我在一起你有多委屈似的！

难道你忘了当初主动攀附的人是你吗？

妈妈敲了敲门，提着给女儿新买的包包进屋了。

“别哭了乖乖，看看你之前订的包包来了。”她把包丢到床上，又抽了两张抽纸坐到Vie旁边给她擦眼泪：“好了好了，都苦丑了，不好看了。”

“我才没哭。”Vie擤了一把鼻涕。

“跟妈妈说说是怎么回事，是Nohm惹你生气了吗？”妈妈宠你地拍拍女儿的背，看她哭的样子和小时候受委屈哭的表情一模一样。

Vie哭得梨花带雨：“他给我写了信，说要跟我分手，要跟公司解绑。”她又继续说：“这两年我给了他那么多资源，带他演戏，给他拍MV，当时也是他先跟我表白追我的，现在却来搞这么一出。”

妈妈心疼女儿，又替她生气不值，她问道：“原因是什么呢？他喜欢上其他女生了吗？”

Vie闻言更委屈了，一边点头一边嚎啕大哭：“嗯。”

妈妈把女儿抱到怀里：“我的宝贝不要哭，没事的乖乖宝贝。这个事情交给妈妈处理好不好，妈妈帮你出这口气。”

她小心呵护着怀里的美玉，试图耗尽世间的奇珍异宝，修复其间如米粒般细小的裂缝。

Vie哭得有些迷糊：“不要妈妈。”她坐直了身子：“我要自己处理这个事情，我可以处理好的。”

“好。”妈妈拍拍她的肩膀：“我的乖女儿，需要什么人就跟我说，妈妈帮你安排。”

安抚女儿睡下后，妈妈回到了房间。

爸爸正在办公桌旁整理企业计划，见老婆来了于是摘下眼镜问道女儿的情况。

“你放手让Vie自己去处理，不怕她闯出什么祸吗？”爸爸知道女儿的辣椒脾气，怕她控制不好尺度。

妈妈擦着护手霜躺到床上：“所以我说让她需要什么人来找我，我给她安排。”

爸爸点点头：“也好，这样你也能把控一下，不至于生出事端。”

“你就放心做好公司的事情吧，剩下的我来安排。”她习惯了幕后操盘手的角色，在她似有似无的干预下，公司和家庭几十年来没有出过任何问题。

时间来到了三月，春意盎然，合作的新剧刚杀青，Nohm就收到了自己的专辑入围，自己会拿奖的消息。

Vie本来不必去观礼，但她已经迫不及待近距离看这出她作为导演呈现的好戏了。

这边嘉宾纷至沓来，优雅落座，纸醉金迷，乍一看好似天宫盛宴，精妙绝伦，不似人间。男的身着各色西装，女的身披各款绫罗，好一群绰约仙子在夜里偷欢。

Vie端坐着，曲线傲人，身旁Nohm翘着二郎腿，一副打了胜仗的样子。

“听说你待会要去领奖？”Vie明知故问道。

Nohm也不谦虚：“如果太阳是从东边升起的话就是的。”

“真是大歌手的做派，讲话都变得那么傲气了。”她理了理自己的裙摆，昔日爱人一改往日的胆怯和沉默，倒让她有些不自在。

“因为爱会让人骄傲。”他转头看着Vie说道：“薇薇公主，这不就是你一直骄傲的原因吗？”

她心里突然生出被人戳破真相的窘迫，随即又被仇恨掩盖：“好啊，那我就等着看你能骄傲到什么时候。”

她把嘴贴到Nohm耳边，用极具威胁的声音说道：“小心一点，借我东西，都得还的。”

Nohm双手抱膝，回道：“如果你觉得是借的，那就拿回去；拿不回去的，就证明是我的。”

是啊，你觉得你给了我影视剧资源，那我以不拍戏还你。

那现在你要把我的嗓子从身体里剥离开吗？

如此强盗逻辑，真叫Vie无语。

同样是夜，白杨所见的就是万鸟归巢，静谧平和。

她抱着笔记本电脑半躺在客厅沙发上，后面就是闪着幽幽蓝光的泳池，正前方就是大门。她想等Nohm办完事回家的时候，第一眼就能看到有人在等他。

白杨的手指不断在电脑上打字，时不时打开新的界面查询资料，偶尔抱怨几句难写的论文，难读的书。

晚风吹过她的后颈，依然有些温热。她摸了摸自己的脖子，突然觉得有些不对：我不是把窗户都关好了吗？怎么会有热风跑进来？

她把电脑放到一边，准备到后面看看是怎么回事。

霎时间，一只大手捂住她的嘴巴，另外一只手拖着她的后脑勺，把她整个人往后面拉。

白杨顿时感觉天旋地转，根本来不及反应，她的手脚不断地挥舞着，想要抓点什么东西保持最基本的平衡，但她的身体已经快被拉扯变形了。

等她觉得自己已经稳定的时候，才发现自己的脚还搭在沙发上，后背已经贴到地面了，整个人呈现出一个诡异的角度。

她的嘴依然被捂着。

一睁眼，只看到两个带着黑色面罩的头。

是抢劫的吗？白杨的大脑飞速运转：只要钱的话，什么都好说。

不等她再想些什么，另一个人直接从腰后掏出两根尼龙绳，不到半分钟就把她的手和脚分别捆好了，随即一屁股坐到了沙发上，大功告成。

白杨没有挣扎，只是默默观察着两个人的动作和所有细节，希望把所有的线索都装在脑袋里。

可是他俩一身黑衣，连一点点配饰或者是纹身都没露出来。

她的心快提到了嗓子眼，还不安分地跳动着，感觉下一秒就要从她嘴里跳出来了。

显然这个抢劫犯并不愿意看到这样的局面，于是掏出了一块毛巾，把她的嘴塞得死死的，然后像提溜小鸡仔一样把她丢回到沙发上，自己则坐到了另一个抢劫犯的身边。

白杨的豆大的眼泪不断滴落，尽管她一直试图强行让自己淡定一点，但还是难以自制。她逼着自己观察这两个人，发现一个人的个子稍微高一点，大概175cm左右，另一个大约170cm，稍微有点小肚子。

由于俩人是坐着的，白杨怕自己估计的数值不准，又用目光搜寻着有用的参照物，比较从自己的角度出发，二人的身高离各种参照物的距离。

一高一矮两个人歇了一会，又对着电视机一顿比划，随即开始翻箱倒柜找东西。

如果他们要钱，为什么不上楼从卧室开始找？

白杨努力调整着自己的呼吸，她一直猜测着二人此行的目的，直到看到他俩从柜子里翻出了遥控器，准备开电视，结果电视机一直没反应，两个人又当起了维修工，准备把电视机后面的线接上。

虽然局势更加不明朗了，但好在一时半会还死不了。

“你都不看电视吗？”高个子的人突然用泰式英语说了一句。

白杨愣住了。

“我说，你不看电视吗？”还是那个高个子的人，一边晃悠着电视机线，一边说着这句莫名其妙的话。

白杨深吸了一口气，试探着摇摇头。

那两人不约而同的冷笑一声，吓得白杨直发毛，随即他们互相交谈着打开电视，手指一直按着遥控器换台。

白杨认真观察着，希望记下他们的音色。

“诶诶诶。”矮个子的人一边说着，一边指着电视。白杨顺着他手指的方向一看，电视上竟然出现了Nohm的脸。

他穿着白色的短袖校服，坐在阶梯教室里，在老师的眼皮子底下和旁边的同学有说有笑。

白杨瞪大了眼睛，上半身不自觉地往前倾：Nohm怎么会在电视里？是什么电视剧吗？

他不是搞音乐的吗？怎么会去演戏？

白杨持续沉浸在震惊中，直到被那二人的嗤笑声打断。

白杨试探着慢慢抬起手，在二人的注视下扯掉嘴里的毛巾，见那两人并没有阻止的意思，她鼓起勇气开口用英语问道：“你们想要什么？”

“想要钱的话，去楼上卧室找，床头柜里有很多现金，也有欧元。”

“钱包里也有泰国的银行卡，密码在我手机里，我可以找出来告诉你们。”

她颤抖着筑起自卫的篱笆，只求能有一线生机。不管他们要什么，保全自己的性命才是最重要的。

高个子的绑匪拿起她放在茶几上的手机问道：“小妞，你对目前的情况一无所知呢对吧？”

白杨捏紧了手里的毛巾，汗水止不住地流，仿佛再说一句话就要爆炸了。但这句话她必须说，因为这关系道如果她能活下来，能不能协助警方捉到这两个犯人。

“我......”白杨的手指甲快嵌入骨肉里了：“我听不太懂你说的话，你可以写下来吗？”

只要有了笔迹，起码可以做笔迹对比。

偌大的房间里只能听见矮个子男不断转换电视机频道的声音。

高个子男抽出腰间别着的匕首，慢慢走向她。白杨知道自己玩砸了，便一个劲儿地往沙发角落里缩。

“不要跟我玩这些小把戏。”他把冰冷的匕首贴到她的脸上，瞬间匕首就被由极度惊惧所产生的汗液包围了：“知道了吗？”

白杨闭着眼睛，连呼吸都忘了，只顾着连连点头。

胖子男招呼了一声，白杨感受到匕首的温度慢慢远离了，便转过身背对着他们，睁开眼睛使劲喘气。

“来吧，好戏开始了。”瘦子男拍了拍她的肩膀，瞬间白杨的身子如同铅块一样重。

“我们来玩个游戏吧。”他把白杨的脸扭过来，正对着电视机：“你给他打电话，如果他接了，我就考虑放过你，可以吗？”

电视里好像正播放着一场盛大的颁奖典礼，可是这台上两个主持人，一男一女，她一个都不认识啊。

等等，在一圈泰语之中，她看到大屏幕上放着“Nohm”的名字，旁边还配了他的照片。

观众席响起掌声，摄像机切换到那张稚嫩而精致的脸庞。少年扣好墨蓝色西服，站起身来，意气风发地向观众席致敬，和身旁的好友握手拥抱......

而坐在他旁边的人，白杨是认得的。尽管只在餐厅有过一面之缘，但Vie的美丽足以让所有人过目不忘。

电视机里歌舞升平，这是属于他的盛典狂欢。

电视机外诡谲莫测，两只野猫不断挑弄着这只仓鼠。
